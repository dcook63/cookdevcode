Alaina's Projects

Med-Assist

Alaina worked with a team of four other students at Kennesaw State University to create her senior project, Med-Assist. Med-Assist is a telemedicine web application that helps medical professionals connect with patients to provide better care out of the doctor' office. The application manages patient appointments, personal information, prescription refill requests, and any medical forms and reports. Additionally, the application provides a doctor-to-patient chat system for quick interaction between the doctor and the patient.

Accomplishments
- Designed application mockups
- Developed the UI of the application
- Designed and created an AWS S3 repository for reports and forms
- Maintained a Kanban board with tasks for the development team
- Presented the project for Kennesaw State University's Industry Advisory Board

Critical Wakeup

Alaina worked with five other students at Kennesaw State University to developer the Android app Critical Wakeup. Critical Wakeup is an alarm app that, based on a certain critical level set by the user, displays puzzles of varying difficulty that must be solved to turn off the alarm. THis app can be useful for people who have trouble waking up in the mornings or have important deadlines they're afraid of sleeping through.

Accomplishments
- Designed app mockups for all screens
- Created front end for Android app
- Gave presentations to class and professor for using app
